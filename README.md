# Nota

A minimalist note-taking app that runs as a single HTML file. No server, no install, no account — just open and write.

## Features

- **Offline first** — works entirely in your browser, no internet connection required
- **Markdown support** — write in plain Markdown with a live split preview
- **Persistent storage** — notes are saved automatically to your browser's local storage
- **Markdown cheatsheet** — built-in reference panel you can toggle while writing
- **Search** — filter notes instantly as you type
- **Word count** — live word, character, and line count in the status bar
- **Three view modes** — edit, split, and preview


## Tech

Built with vanilla HTML, CSS, and JavaScript.
Uses [marked.js](https://marked.js.org/) for Markdown rendering and browser `localStorage` for saving notes.
